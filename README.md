# Employee Map Dashboard

by Szymon Stasiak

### Install packages

```
npm install
```

### Start application

```
npm run dev
```

### Build static files

```
npm run build
```

### Build and run docker image

## Make sure docker application is running

```
docker build -t <image name> .
docker run -p 80:80 <image name>
```
